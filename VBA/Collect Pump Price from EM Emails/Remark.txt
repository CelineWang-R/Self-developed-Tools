Purpose: To automate the extraction of pump price data from daily system-generated emails


Latest version: ThisOutlookSession_V1
Date Modified: 2025-11-14 4:00PM

Process:
1. Open VBA editor in Outlook. Then copy and paste the code from Line 10
2. Modify 1/Excel output path -- Line 57
		  2/Date filter bound -- Line 60
		  3/Mailbox Name -- Line 21, 64
		  4/Sender filter -- Line 29, 74
		  5/Subject filter -- Line 28, 75
	if needed
	(Please make sure the auto save is turne on if the file is stored in a OneDrive folder)
3. Save the macro and restart Outlook
4. If the user wants to collect data from all the previous emails, run the macro manually


Logic:
The macro iterates through all emails that meet the specified conditions and extracts the price information.
Emails that have already been processed are marked with the category 'Processed', ensuring they are skipped in subsequent runs.
The macro will run automatically when the new qualified email arrives.
There will be a message box each time the macro runs. 
**** Please be cautious as the automatic trigger will fail when Excel Application is open. ****

**** Please input the monthly price of LPG manually. ****
