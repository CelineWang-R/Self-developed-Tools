Files included:
1. csv_to_xlsx.bat

How to use:
- Put this BAT file into the folder that contains your CSV files.
- Double-click csv_to_xlsx.bat
- It will convert every .csv file in that folder into .xlsx
- Output files go into the subfolder: result_xlsx

Notes:
- It uses embedded Python code inside the BAT file
- Original CSV files are not changed

Requirements:
- Python installed
- Packages:
  pip install openpyxl
